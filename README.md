# git-tutorial

### Создать папку
```
mkdir dir1
```
### Удалить папку
```
rmdir dir1
```
### Создать папку с содержимым
```
mkdir -p dir1/dir/2/dir3
```
### Удалить папку с содержимым
```
rm -r dir1
```
### Создать пустой файл
```
touch file.txt
```
### Создать / Перезаписать файл с содержимым
```
ehco '123' > file.txt
```
